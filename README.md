# Syntaxable

Modern collaborative coding platform with AI assistance, realtime editing, practice problems, and social features.

![React](https://img.shields.io/badge/React-18.3-61DAFB?logo=react) ![TypeScript](https://img.shields.io/badge/TypeScript-5.5-3178C6?logo=typescript) ![Vite](https://img.shields.io/badge/Vite-6.3-646CFF?logo=vite) ![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4-06B6D4?logo=tailwindcss) ![Supabase](https://img.shields.io/badge/Supabase-Database-3ECF8E?logo=supabase)

## Highlights

- **Collaborative editor**: realtime presence, files/folders, version history
- **AI assistant**: context-aware chat + persistent history
- **Practice**: courses, problems, batch test runner, progress tracking
- **Community**: profiles, friends, project sharing

## Quickstart

```bash
npm install
npm run dev

```

## Stack

- **Frontend**: React + TypeScript + Vite + Tailwind
- **Backend**: Supabase (Postgres, Auth, Realtime, Edge Functions)
